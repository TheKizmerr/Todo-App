Przed startem projektu trzeba zainstalować node_modules z React (npm install react-scripts --save)

Do projektu użyto technologii:
HTML, CSS, JS & React, Bootstrap

Pisanie kodu zostało oparte o syntax przyjęty globalnie dla biblioteki React (PascalCase dla nazw komponentów itd)

Głównym moim zadaniem było pokazanie mechaniki i implementacji jezyka JavaScript w React.

Pomocy szukałem w źródłach PluralSight, lecz głównie polegałem na dokumentacji np. W3School, MDN Web Docs oraz StackOverflow oraz książke Marjin Haverbejke "Eloquent JavaScript".
